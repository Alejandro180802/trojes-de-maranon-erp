# Roadmap de entregas

1. Fundación: autenticación, roles, catálogos, proyectos/plataformas, Storage y shell.
2. Operación: reportes diarios, inventario, entradas/salidas/conteos y reversos.
3. Consumo y equipo: estimados, maquinaria, horómetros y diésel.
4. Continuidad: mantenimiento, alertas, dashboard y exportaciones.
5. Piloto: datos iniciales, capacitación y validación en obra.
